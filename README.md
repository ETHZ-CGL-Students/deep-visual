# Deep Visual

Visualizing deep neural networks

Bachelor Thesis, Marco Crespi, 2018 @ ETHZ

## Setup

### Python (backend)

1.  `pip3 install tensorflow keras eventlet flask flask_socketio`

### React (frontend)

1.  `cd app`
1.  `npm install`

## Run

Start two command prompts

- `python3 test2.py`
- `cd app && npm start`
